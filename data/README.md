## Instructions pour télécharger les données

Les fichiers de données doivent être téléchargés depuis [data.gouv.fr](https://data.gouv.fr/) et placés dans ce dossier.

### Fichiers requis

1. **vacsi-tot-a-dep-2023-07-13-15h50.csv**
   - Données de couverture vaccinale par âge et département
   - Accessible via la plateforme Santé Publique France sur data.gouv.fr

2. **vacsi-tot-s-dep-2023-07-13-15h51.csv**
   - Données de couverture vaccinale par sexe et département
   - Accessible via la plateforme Santé Publique France sur data.gouv.fr

### Étapes de téléchargement

1. Aller sur [data.gouv.fr](https://data.gouv.fr/)
2. Rechercher "VACSI" (Vaccination COVID-19 par région et département)
3. Télécharger les fichiers CSV correspondants aux dates mentionnées
4. Placer les fichiers dans ce dossier (`data/`)

## Instructions for downloading data

Data files must be downloaded from [data.gouv.fr](https://data.gouv.fr/) and placed in this folder.

### Required files

1. **vacsi-tot-a-dep-2023-07-13-15h50.csv**
   - Vaccination coverage data by age and department
   - Available from Public Health France platform on data.gouv.fr

2. **vacsi-tot-s-dep-2023-07-13-15h51.csv**
   - Vaccination coverage data by gender and department
   - Available from Public Health France platform on data.gouv.fr

### Download steps

1. Go to [data.gouv.fr](https://data.gouv.fr/)
2. Search for "VACSI" (COVID-19 Vaccination by region and department)
3. Download the CSV files corresponding to the mentioned dates
4. Place the files in this folder (`data/`)

**Note**: The application will look for these files in the same directory as `app_streamlit.py`, so make sure they are accessible.
